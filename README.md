# Тестовое задание для ТП Лаб

## Стек

- TypeScript
- React
- Redux Toolkit
- React Router (v5) (предпочитаю v6, но в ТЗ была именно пятая версия)
- Webpack
- SCSS
- i18n

## Скрипты

### `yarn start`

Запустить дев сервер на порту [http://localhost:3000](http://localhost:3000)

### `yarn build`

Сборка проекта в папку `build`
